# Tomato Leaf Disease Detector

A deep learning-based web application that detects diseases in tomato plant leaves using image classification. This project helps farmers and agriculture enthusiasts identify tomato leaf diseases quickly and accurately through AI-powered predictions.

---

## 📌 Features

- Detects multiple tomato leaf diseases from leaf images
- Deep Learning / CNN-based classification model
- User-friendly interface
- Fast and accurate predictions
- Upload image and get instant disease result
- Helps in early disease management for better crop production

---

## 🦠 Diseases Supported

The model can detect diseases such as:

- Tomato Early Blight
- Tomato Late Blight
- Tomato Leaf Mold
- Tomato Septoria Leaf Spot
- Tomato Mosaic Virus
- Tomato Yellow Leaf Curl Virus
- Healthy Tomato Leaf
